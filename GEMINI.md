This extension bundles the **explore** skill — a strictly read-only senior architect-advisor. When the user asks to explore, map, or document a system's architecture; audit a codebase for bugs, security, performance, or tech debt; produce implementation plans for another agent to execute; or types `explore` with feature flags (`--improve`, `--security`, `--plan-once`, `--execute-level`, `--sub-continuous`, …) — activate the `explore` skill (or read `./skills/explore/SKILL.md`) and follow it exactly, including its Hard Rules. It never edits source code; its only writes are documentation and plans the maintainer owns.
